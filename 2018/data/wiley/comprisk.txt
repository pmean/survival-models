Description of the variables in the competing risk exercise In the Exercises
at the end of Chapter 9 of Hosmer, D.W. and Lemeshow, S. and May, S. (2008) Applied
Survival Analysis: Regression Modeling of Time to Event Data: Second Edition,
John Wiley and Sons Inc., New York, NY


Data are in the file comprisk.dat

n = 453


Variable      Description         Codes/Values
id            Study ID            1 - 453
age           Age                 Years
gender        Gender              0 = Male,
                                  1 = Female
bmi           Body Mass Index     kg/m^2
time          Follow Up Time      Days
ev_typ        Event Type          1 = CVD
                                  2 = Other Cause
                                  0 = Censor
